Minicursos

  • Minicurso 1: Grails - Clésio Gonçalves

    • Grails é um framework para construção de aplicações para web através da linguagem de programação Groovy (uma linguagem dinâmica para a plataforma Java). Foi desenvolvido em código aberto e pretende ser um framework de alta produtividade graças à utilização do paradigma da programação por convenção que preserva o desenvolvedor dos detalhes de configuração.

  • Minicurso 2: CorelDraw - Danilo Lopes

    • O CorelDraw é um aplicativo de ilustração vetorial e layout de página que possibilita a criação e a manipulação de vários produtos, como por exemplo: desenhos artísticos, publicitários, logotipos, capas de revistas, livros, CDs, etc.

  • Minicurso 3: BootStrap - Aroldo

    • Bootstrap é um framework front-end de código aberto (opensource). Em palavras simples, é um conjunto de ferramentas criadas para facilitar o desenvolvimento de sites e sistemas web, compatível com HTML5 e CSS3.

  • Minicurso 4: PhotoShop - Danilo Lopes

    • Software caracterizado como editor de imagens bidimensionais. É considerado o líder no mercado dos editores de imagem profissionais, assim como o programa é de fato para edição profissional de imagens digitais e trabalhos de pré-impressão.

  • Minicurso 5: Django - Jonnison Lima

    • Django é um framework web de alto nível escrito em Python que estimula o desenvolvimento rápido e limpo. Ele permite que você construa rapidamente aplicações web de alto desempenho e elegância.

  • Minicurso 6: Cake-PHP - Woshington Valdeci

    • CakePHP é um framework escrito em PHP que tem como principais objetivos oferecer uma estrutura que possibilite aos programadores de PHP de todos os níveis desenvolverem aplicações robustas rapidamente, sem perder flexibilidade.
